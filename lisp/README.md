# Lisp

## Install

sudo apt install sbcl

sudo apt install clisp

## Common Command

### Exit REPL

(exit)

### Run

clisp hello.lisp

### Compile

clisp -c hello.lisp

clisp hello.fas (bytecode)

(compile-file "hello.lisp")
